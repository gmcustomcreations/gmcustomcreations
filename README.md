# G M Custom Creations Website

Premium portfolio for bespoke woodworking.

## Auto Gallery Builder
After placing images in `images/[category]/` folders, run:

```bash
node generate-gallery.js
```

This auto-scans and updates the JS array in index.html.

Refresh the page — new work appears instantly.

## View the Website
Open `index.html` in your browser.

**Logo**: Place your logo file as `images/logo.png` (recommended size ~200x80px, transparent PNG). The header will display it automatically.

For a public link, upload to Netlify/Vercel/GitHub Pages.
